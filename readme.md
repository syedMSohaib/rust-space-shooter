# Rust Space Shooter

> A simple yet cool space shooter game built on Rust, Cargo and Bevy

## Install

```
$ git clone https://github.com/syedMSohaib/rust-space-shooter
$ cd rust-space-shooter
$ cargo build
$ cargo run
```
